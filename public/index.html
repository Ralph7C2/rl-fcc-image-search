Image Search Abstraction Layer

Description -

Conventiently get the URL, alt text, and thumbnails for an image search, paginated.

Usage:

Get image search info for <TERM>: https://rl-fcc-image-search.herokuapps.com/api/imagesearch/<TERM>

Get the next set of results(Starting with the 11th item): https://rl-fcc-image-search.herokuapps.com/api/imagesearch/<TERM>?offset=11

Get the second page(pages of 10): https://rl-fcc-image-search.herokuapps.com/api/imagesearch/<TERM>?page=2

Get a list of the ten most recent terms searched: https://rl-fcc-image-search.herokuapps.com/api/latest
